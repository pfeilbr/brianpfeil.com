# Alpha Sound

<p><img src="/ios-apps/alpha-sound/images/app-icon-alt-d-114x114.png" alt="" /></p>

<p><strong>Alpha Sound is a simple ABC&rsquo;s learning program for children. Children learn
    their letters by seeing and hearing the letter names.</strong></p>

<div style="clear: both"></div>


<h2>Features</h2>

<ul>
  <li>Hear the name of the letter</li>
  <li>Uppercase and lowercase letters</li>
  <li>Quickly access any letter</li>
  <li>Autoplay letters</li>
  <li>Large easy to read letters</li>
  <li>Includes numbers 0 thru 10</li>
</ul>
