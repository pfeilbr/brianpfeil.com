## [Mocha](http://mochajs.org/) Playground

Area to learn and explore the [Mocha](http://mochajs.org/) JavaScript test framework

### Install Mocha

    $ npm install -g mocha

### Install THIS and dependencies

Clone repo then

    $ npm install

### Run Tests

    $ mocha
